# CLAUDE.md

이 파일은 Claude Code가 이 저장소에서 작업할 때 참고하는 가이드다.

## 프로젝트 한 줄 요약
레미콘 운전기사용 개인 운행일지 PWA. Vanilla JS(ES5) + localStorage. 빌드 없음.

## 작업 시 반드시 지킬 것

1. **ES5 문법 유지**: `var`/`function`만 사용. 화살표 함수, `const`/`let`, 템플릿 리터럴 도입하지 말 것 (구형 모바일 호환).
2. **금액은 절대 반올림/절삭 금지**: 항상 `toLocaleString()`로 전체 금액 표시.
3. **기본급 = 바리수 × unitPrice** (바리당 단가). 근무일수와 곱하지 말 것.
4. **유류 부호**: `소비량 - 주유량 >= 0` 이면 "지급 받음"(초록), 음수면 "도급비 차감"(빨강). 운전기사가 받는 돈 기준.
5. **유류는 월별 리셋**: 이전 달 이월 없음. 월말/연말 결산 금액에 유류 포함하지 말 것.
6. **2시간초과 정산완료(settled) 건은 결산에서 제외**.
7. HTML 문자열은 `+` concat으로 생성됨. 수정 시 따옴표 이스케이프(`\'`) 주의.

## 파일 역할
- `index.html` — 뼈대. `<nav>` 탭 버튼 + `#mc` 컨테이너 + CSS/JS 링크.
- `css/style.css` — `--th-*` 테마 변수 + 전체 스타일. 색상은 하드코딩 말고 변수 사용.
- `js/app.js` — 전 기능. 전역 상태(`tab,cd,sel,logs,cfg`...) + 렌더 함수(`rCal/rEntry/rStats/rAnnual/rConfig`).

## 렌더링 흐름
`go(tab)` → `render()` → 탭별 `rXxx()` 함수가 `#mc.innerHTML` 갱신.
달력 탭일 때만 `#mc`에 `cal-mode` 클래스 추가(꽉찬 레이아웃).

## 데이터 키
- `rl9_logs`, `rl9_cfg` (구버전 `rl8_*` 폴백). 스키마 변경 시 마이그레이션 코드 추가하고 키 버전 올릴 것.

## 로컬 실행
`python3 -m http.server 8000` 후 브라우저 접속. 빌드/설치 단계 없음.

## 테스트 시 주의
- localStorage 기반이라 시크릿창/캐시삭제로 데이터 초기화됨.
- 모바일 레이아웃(safe-area, 홈 인디케이터)은 실기기 또는 기기 시뮬레이션으로 확인.
